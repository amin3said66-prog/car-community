# Car Community

An Angular 20 community platform for car enthusiasts. Features car listings, community posts, events, member directory, and an admin dashboard. Uses mock data — no backend or external services required.

## Stack

- **Angular 20** (standalone components)
- **Angular Material 20**
- **Tailwind CSS 3**
- **TypeScript 5.6+**
- **Node.js 20** (required — Angular CLI 20 needs ≥ v20.19)

## How to run

```bash
npm install --legacy-peer-deps
npm start
```

The app serves on port 5000 (`ng serve --host 0.0.0.0 --poll 2000 --port 5000`).

## User preferences

- No backend/database — all data is mocked via Angular services.
